## À propos

Ce projet a été réalisé dans le cadre de ma formation pratique en **Développement Web** chez FHC Groupe Sarl, complétée par des parcours autodidactes sur **freeCodeCamp**. Il s'agit d'une version web de mon CV, pensée comme un exercice pratique d'HTML sémantique et de mise en page CSS (Flexbox, responsive design).

## Fonctionnalités

- Présentation structurée : Projets, Formation, Compétences, Centres d'intérêt, Langues
- Mise en page responsive (adaptation mobile via media queries)
- Animation de zoom subtile, désactivée automatiquement pour les utilisateurs ayant activé `prefers-reduced-motion`
- Design sombre avec effet de flou (`backdrop-filter`) sur fond illustré

## Technologies utilisées

- **HTML5** sémantique
- **CSS3** (Flexbox, animations, media queries)

## Installation / Utilisation

Aucune dépendance ni serveur nécessaire :

```bash
git clone https://github.com/ton-username/cv-web.git
cd cv-web
```

Puis ouvrir `index.html` directement dans un navigateur.
